generate_scale (root_num, scale_type[, octaves])
Generate scale from a root note.

Parameters:

  • root_num MIDI note number (0-127) where scale will begin.
  • scale_type String defining scale type (eg, "major", "aeolian" or "neapolitan major"), see class for full list.
  • octaves Number of octaves to return, defaults to 1. (optional)

Returns:

    Array of MIDI note numbers.
snap_notes_to_array (note_nums, snap_array)
Snap MIDI note numbers to an array of note numbers.

Parameters:

  • note_nums MIDI note number (0-127) or array of note numbers.
  • snap_array Array of note numbers to snap to, must be in low to high order.

Returns:

    Array of adjusted note numbers or a single note number.
note_nums_to_names (note_nums[, include_octave])
Convert MIDI note numbers to names.

Parameters:

  • note_nums MIDI note number (0-127) or array of note numbers.
  • include_octave Include octave number in return string if set to true. (optional)

Returns:

    Name string (eg, "C#3") or array of strings.
